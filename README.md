# Skills

Agent skills collection for AI coding agents, managed via [agent-skills-nix](https://github.com/Kyure-A/agent-skills-nix).

## Usage

This repository is consumed as a flake input by [dotnix](https://github.com/yutakobayashidev/dotnix). Skills are deployed to `~/.agents/skills`, `~/.config/claude/skills`, and `~/.config/codex/skills` via home-manager.
